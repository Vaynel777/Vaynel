 QUICK START (DEPLOYMENT)
================================================================================

1. PUSH TO GITHUB
   git remote add origin https://github.com/YOUR_USERNAME/openclaw-mission-control.git
   git push -u origin main

2. DEPLOY TO VERCEL
   npm install -g vercel
   vercel --prod
   → Gets instant live URL

3. CONFIGURE MASTER
   export MASTER_HOST="<your-vercel-url>"
   export MASTER_PORT="443"

4. DEPLOY ADMIN DASHBOARDS
   bash admin_dashboard/deploy.sh neo-toshiba 192.168.1.33 <your-vercel-url> 443
   bash admin_dashboard/deploy.sh mba2 192.168.1.56 <your-vercel-url> 443

5. SET UP METRIC SYNC
   On Master VM, add to crontab:
   */5 * * * * curl -s http://192.168.1.33:8001/metrics | \
     curl -X POST -d @- https://<your-vercel-url>/api/metrics/neo-toshiba
